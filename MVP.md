# MVP

### Core Backend:
1. Установить Python + фреймворки.

2. Создать модели:
   - Product (name, price, description, image)
   - User (login, password)
   - Order

4. Интеграция с БД:
   - Использовать SQLAlchemy для создания таблиц.

### Basic Frontend:
1. Использовать Jinja2 для создания основных страниц (главная, корзина, заказ)
